# Configurações de Inicialização Salvas

## Comandos Rápidos de Inicialização

### 1. Iniciar Aplicação
```bash
npm run dev
```

### 2. Push do Schema para Banco
```bash
npm run db:push
```

### 3. Verificar Status do Banco
```bash
npm run db:studio
```
